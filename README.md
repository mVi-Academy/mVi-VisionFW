# mVi-VisionFW
mVi-VisionFW是 mVi Academy基于LabVIEW消息队列状态机(QMS)开发的一套开源机器视觉应用程序框架。该框架不仅可为大多数典型机器视觉项目提供框架支持，也可与mVi-MultiLoopFW多任务框架集成，完成大型复杂的机器视觉任务。
